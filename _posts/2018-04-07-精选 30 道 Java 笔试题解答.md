## 1. 下面哪些是Thread类的方法（）
A start()       
B run()       
C exit()       
D getPriority()  

答案：ABD

解析：看 Java API docs 吧：http://docs.oracle.com/javase/7/docs/api/ ，exit() 是System 类的方法，如 System.exit(0)。
