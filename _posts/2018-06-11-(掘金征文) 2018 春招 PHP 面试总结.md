## PHP

### 基础

**1、session 和 cookie 的区别**

- Session 是在服务端保存的一个数据结构，用来跟踪用户的状态，这个数据可以保存在服务器内存、缓存、数据库中

- Cookie 是客户端保存用户信息的一种机制，用来记录用户的一些信息，也是实现Session 的一种方式。

为什么需要 session ？因为 http 是无状态的，所以需要 session 来记录用户的状态
